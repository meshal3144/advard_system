السياسة والخصوصية

مقدمة

في Advard، نلتزم بحماية بيانات عملائنا وضمان أقصى درجات الأمان والخصوصية. توضح هذه السياسة كيفية جمع بيانات المستخدمين، استخدامها، ومشاركتها وفقًا لأفضل الممارسات والمعايير الأمنية.

جمع المعلومات

نقوم بجمع البيانات التالية عند استخدامك لنظامنا:

  • معلومات الحساب مثل الاسم، البريد الإلكتروني، ورقم الهوية.
  • بيانات الحضور والانصراف لتقديم التقارير والتحليلات.
  • معلومات تقنية مثل عنوان IP ونوع الجهاز المستخدم لضمان الأمان وتحسين الخدمة.

استخدام البيانات

  • تحسين أداء النظام وتحليل تجربة المستخدم
  • توفير تقارير دقيقة عن الحضور والأداء
  • إرسال إشعارات ذكية وفقًا لإعدادات المستخدمين

حماية البيانات

نحن ملتزمون بحماية بياناتك من الوصول غير المصرح به باستخدام تشفير متقدم وإجراءات أمان حديثة. كما أننا لا نشارك بيانات المستخدمين مع أي جهات خارجية بدون موافقتهم.

حقوق المستخدمين

يمكنك طلب تعديل أو حذف بياناتك من خلال إعدادات الحساب أو التواصل مع فريق الدعم الفني لدينا.
